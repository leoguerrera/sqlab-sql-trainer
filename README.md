# SQLab — SQL Trainer

Ambiente interativo de treinamento SQL rodando direto no navegador — sem instalação.

**[▶ Acessar o SQLab](https://leoguerrera.github.io/sqlab-sql-trainer/)**

![SQLab interface](screenshot.jpg)

---

## O que você pode fazer

- Escrever e executar queries SQL diretamente no navegador
- Praticar os principais comandos: SELECT, WHERE, GROUP BY, JOIN, ORDER BY
- Testar lógica SQL sem precisar instalar banco de dados

---

## Tecnologias

- HTML5 / CSS3 / JavaScript puro — sem frameworks externos
- Deploy via GitHub Pages

---

## Contexto

Ferramenta criada durante os estudos do curso SQL Impressionador (Hashtag Treinamentos, 90h).
Desenvolvida em parceria com Claude (Anthropic) — estrutura, iterações e ajustes aplicados pelo autor.
