# SQLab — SQL Trainer

Ambiente interativo de treinamento SQL construído com HTML/JS puro.

**[▶ Acessar o SQLab](https://leoguerrera.github.io/sqlab-sql-trainer/)**

## Sobre
Ferramenta desenvolvida como ambiente de estudo prático de SQL,
simulando queries e exercícios diretamente no navegador — sem instalação.

## Tecnologias
- HTML5 / CSS3 / JavaScript
- Interface construída do zero, sem frameworks externos

## Contexto
Projeto criado durante estudos do curso SQL Impressionador (Hashtag Treinamentos, 90h).
Desenvolvido em parceria com Claude (Anthropic) — conceito, estrutura e iterações feitas
em conjunto, com ajustes e validações aplicados pelo autor.
